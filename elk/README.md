# ELK - ELK

Bring up without keycloak-proxy
`docker-compose up -d elk`

Bring up with keycloak-proxy (all services)
`docker-compose up -d`

Bring down
`docker-compose down`

Bring down and delete all named volumes
`docker-compose down -v`
